# Rust exercises

## Installation

See https://doc.rust-lang.org/book/ch01-01-installation.html

## Setup

```
git clone git@github.com:ericrasmussen/rustexercises.git
cd rust-exercises
```

### add your code!

`$EDITOR src/exercises1.rs`

### run the tests

`cargo test`

### run the `main` function

`cargo run`
